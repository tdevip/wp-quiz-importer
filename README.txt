WP Quiz Importer plugin can import MS Word quiz questions into wordpress quiz providers such as WpProQuiz, LearnPress and LearnDash.

== Description ==

This is a plugin to import MSWord quiz questions into wordpress quiz providers. Using the provided MSWord template prepare quiz questions. Use macro in the word template to create an XML file. Upload this XML file (using the plugin) and import quiz questions into wordpress quiz provder. Currently WpProQuiz,  LearnPress and LearnDash quiz providers are supported. In future other providers such as Watu, Sensei and many more will be supported. Various types of quiz questions are supported (see provided MSWord template for more details). Quiz questions can contain equations in LaTex format (which can be done using MathType editor in MSWord). They may also contain HTML tags and images. 

The basic features of plugin are:
* 1. Quiz questions and answers can be prepared in an MSWord file
* 2. Various types of questions (depending on quiz provider) can be imported.
* 3. Questions can contain LaTex equations and html tags.
* 4. WpProQuiz, LearnPress and LearnDash quiz providers are supported.

The premiun version supports:
* 1. Message when correct answer is selected
* 2. Message to display when incorrect answer is selected
* 3. Hint to disply for each question
* 4. Different marks depending on answer selected 
* 5. Additional question types
* 6. Additional quiz providers

**NOTE:** Some of the above features are dependent on quiz providers.

== Installation ==

How to Install this plugin:

* 1. Copy the folder `wp-quiz-importer` and its contents to the `/wp-content/plugins/` directory.
* 2. Activate the plugin through the 'Plugins' menu in WordPress.
* 3. A sub-menu called "Quiz Importer" will be created under wordpress settings menu.


== Frequently Asked Questions ==

= How to import quiz questions? =

Upload the XML file created using MSWord template (provided along with this plugin) and click import button.
